There are two ways to query this interface.
The first way is with GET requests and curl. An example: 'curl -X GET http://url_of_this_vm/hosts/<hostname>/<ip>' will return OK for good IPs and NotOK for bad IPs.
The second way is to just test it here in the browser, as basic GET queries can also be triggered here.


There are two types of queries that can be used:
The first query type is to lookup a host, ip combination.
An example: /hosts/<hostname>/<ip><>

The second is a date range query for a given host
An example: /hosts/<hostname>/dates/<from_date>/<to_date>;
